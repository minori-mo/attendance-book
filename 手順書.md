## 1. システム概要

複数クラスの出欠状況を日単位で記録・管理するシステムです。

**主な機能**
- 欠席・遅刻・早退・公欠・忌引の記録（複数同時選択可）
- 授業ごとの出席状況の記録
- 月をまたいだ出席カレンダーの表示
- 特定日の時間割を手動で上書き設定
- 欠席理由・欠席授業の集計
- Excel形式での出力

**画面構成**

| タブ | 内容 |
|------|------|
| 出席入力 | カレンダー形式で出欠を記録する |
| クラス管理 | クラスの追加・編集・削除 |
| 生徒管理 | 生徒の追加・編集・削除 |
| 時間割管理 | 授業の登録・時間割の作成 |

## 2. 初期設定の流れ

はじめて使用する際は、以下の順序で設定してください。

```
① 授業登録  →  ② 時間割の作成  →  ③ クラスの登録  →  ④ 生徒の登録
```

---

## 3. 授業登録

時間割で使用する授業科目を先に登録します。

### 操作手順

1. **「時間割管理」タブ**を開く
2. 左側の「**授業登録**」欄に授業名を入力する（例：国語、数学、英語）
3. **「＋」ボタン**を押すか、Enter キーを押す
4. 使用するすべての授業を繰り返し登録する

### 注意事項

- 同じ名前の授業は登録できません（エラーメッセージが表示されます）
- 授業名は**編集ボタン**から変更できます
- 授業を削除すると、その授業が設定されていた時間割のコマも同時に削除されます

---

## 4. 時間割の作成

曜日・時限ごとに授業を割り当てた時間割を作成します。

### 操作手順

1. **「時間割管理」タブ**を開く
2. 左側「時間割」の**「＋ 追加」ボタン**を押す
3. 時間割名を入力する（例：前期時間割、後期時間割）
4. グリッドの各セルをクリックし、ドロップダウンから授業を選択する
   - 行が「時限（0〜10）」、列が「曜日（月〜日）」
   - 授業なしのコマは「―」のまま
5. **「保存」ボタン**を押す

### ポイント

- 時間割は複数作成できます（前期・後期など用途に応じて使い分けられます）
- 保存後は「✓ 保存しました」と表示されます

---

## 5. クラスの登録

### 操作手順

1. **「クラス管理」タブ**を開く
2. 学年（例：1年）とクラス名（例：A組）を入力する
3. **「追加」ボタン**を押す

---

## 6. 生徒の登録

### 操作手順

1. **「生徒管理」タブ**を開く
2. 上部のドロップダウンで対象**クラスを選択**する
3. 出席番号・氏名を入力して**「追加」ボタン**を押す
4. すべての生徒を繰り返し登録する

### 注意事項

- 出席番号は省略可能です
- 生徒の情報を変更しても、記録済みの出席データには影響しません

## 7. 日々の出席入力

### 7-1. 日付に時間割を設定する

授業ごとの出席を記録するには、先にその日の時間割を設定する必要があります。

1. **「出席入力」タブ**を開く
2. カレンダーの**日付ヘッダー（数字部分）をクリック**する
3. 表示されたポップアップで時間割を選択する
4. 時間割が設定された日付には**青い点（●）**が表示されます

> 時間割を設定しなくても、欠席・遅刻などの記録はできます。

### 7-2. 特定日の時間割を手動で設定する

授業変更などで、その日だけ通常と異なる時間割にしたい場合は手動設定を使います。

1. カレンダーの**日付ヘッダーをクリック**する
2. ポップアップ下部の**「時間割を手動で設定」ボタン**を押す
3. 0〜10時限それぞれの授業をドロップダウンで選択する
4. **「保存」ボタン**を押す

手動設定が保存された日付には**オレンジの点（●）**が表示されます。

| ボタン | 動作 |
|--------|------|
| 保存 | 手動設定した時間割を保存する |
| 手動設定を削除 | 手動設定を削除し、通常の時間割に戻す |
| キャンセル | 変更を破棄して閉じる |

> 手動設定を削除すると、その日の授業出席の記録も合わせて削除されます。

### 7-3. 出欠を記録する

1. 対象生徒・日付の**セルをクリック**してモーダルを開く
2. 「出欠」欄で該当する状況にチェックを入れる

| 区分 | 略称 | 内容 |
|------|------|------|
| 欠席 | 欠 | 終日欠席 |
| 遅刻 | 遅 | 授業開始後に登校 |
| 早退 | 早 | 授業終了前に下校 |
| 公欠 | 公 | 学校行事・部活動など公認の欠席 |
| 忌引 | 忌 | 親族の不幸による欠席 |

- 複数の区分を同時に選択できます（例：遅刻＋早退）
- チェックを外すと記録が削除されます

### 7-4. 授業ごとの出席を記録する

出欠チェックを入れると、モーダル右側に「授業出席」欄が表示されます。

1. 出欠にチェックを入れる
2. 「授業出席」欄が表示されたら、**実際に出席した授業にチェック**を入れる

> 時間割（または手動設定）が設定されていない日は「時間割を設定すると出席した授業を選べます」と表示されます。

### 7-5. 表示範囲の切り替え

画面上部の「表示範囲」で開始月・終了月を変更すると、複数月にまたがるカレンダーを表示できます。

---

## 8. 集計の見方

カレンダー右側に集計列が表示されます（「集計を隠す」ボタンで非表示にできます）。

| 列 | 内容 |
|----|------|
| 欠席理由 | 欠・遅・早・公・忌の回数をバッジで表示 |
| 欠席授業 | 授業別の欠席コマ数を表示（例：国語2） |

---

## 9. Excel出力

1. **「出席入力」タブ**で対象クラスと表示範囲を設定する
2. **「Excel出力」ボタン**を押す
3. ファイルが自動でダウンロードされる

**出力内容**

| シート | 内容 |
|--------|------|
| クラス名（例：1年A組） | 日別の出欠カレンダー |
| 集計 | 生徒ごとの欠席理由・欠席授業のまとめ |
